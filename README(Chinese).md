# 🤖 AI 多智能体量化投研系统

这是一个基于 **Python** + **LangGraph** + **DeepSeek** 开发的自动化股票分析系统。它模拟了一个真实的对冲基金投资委员会，由 5 个专业的 AI 智能体协同工作，通过“辩论”产出最终的投资建议。

## 🌟 这个系统能做什么？

当你输入一个股票代码（如 `sh.600519`）后，系统会启动以下流程：

1. **📈 技术面分析师**：拉取最近 15 天的 K 线图和均线，看趋势。
2. **📊 基本面研究员**：去审计最新的财务报表，看盈利能力。
3. **🌐 市场情绪员**：全网爬取最新新闻和舆情，看大家都在聊什么。
4. **🛡️ 首席风控官 (CRO)**：评估大盘风险和波动率，拥有**一票否决权**。
5. **🏆 投资委员会 (CIO)**：汇总所有人报告，进行多空对决，给出的最终结论（买入/卖出/观望）。

## 🚀 针对新手的快速上手指南

如果你从来没有写过代码，请严格按照以下步骤操作：

### 1. 准备工作 (环境安装)

* **安装 Python**：去 [python.org](https://www.python.org/) 下载并安装 Python 3.8 或更高版本。安装时一定要勾选 **"Add Python to PATH"**。
* **获取 API Key**：去 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并充值（几块钱就行），然后创建一个以 `sk-` 开头的 API Key。

### 2. 获取项目代码

* **下载压缩包**：在 GitHub 页面点击绿色的 **"Code"** 按钮，选择 **"Download ZIP"**。
* **解压**：将下载好的压缩包解压到你的桌面或某个文件夹里。

### 3. 安装运行所需的库 (极其重要)

你需要告诉电脑这个程序运行需要哪些“插件”。

1. 找到你的项目文件夹，在空白处**按住 Shift + 鼠标右键**。
2. 选择 **“在此处打开 PowerShell 窗口”** 或 **“在终端中打开”**。
3. 输入以下命令并回车（这可能需要 1-2 分钟）：
```bash
pip install flask flask-cors baostock pandas numpy langchain-openai langgraph requests duckduckgo-search urllib3

```



### 4. 运行系统 (双端操作)

#### **第一步：启动后端服务 (大脑)**

1. 在刚才的终端窗口输入：
```bash
python app.py

```


2. 如果你看到 `🚀 正在启动多智能体交易 API 服务...` 且没有报错，说明“大脑”已经上线了。**请不要关闭这个窗口！**

#### **第二步：启动前端界面 (脸面)**

1. 在文件夹里找到那个 `智能协作体.html` 文件。
2. **直接双击**它，它会在你的浏览器里打开。

## 💡 如何使用进行分析？

1. **填入 Key**：在网页顶部的第一个框里，粘贴你刚才在 DeepSeek 官网申请的 `sk-xxxx` 密钥。
2. **输入股票代码**：在第二个框里输入代码。
* **A 股**：必须带后缀，如 `sh.600519` (茅台), `sz.000001` (平安)。
* **美股**：直接输入，如 `AAPL` (苹果), `NVDA` (英伟达)。


3. **点击开始**：点击按钮后，稍微等 15-30 秒。你可以切换回那个黑色背景的终端窗口，看 AI 们正在热烈讨论的日志。

## 📂 文件结构说明 (给想学习的你)

* `app.py`：服务器入口，负责把前端和 AI 连接起来。
* `main_workflow.py`：工作流编排，规定了 AI 们谁先说话，谁后说话。
* `sentiment_agent.py`：负责全网搜新闻（情绪面）。
* `tech_agent.py`：负责看图表指标（技术面）。
* `fund_agent.py`：负责看赚钱能力（基本面）。
* `risk_agent.py`：负责盯着风险（风控面）。
* `智能协作体.html`：你操作的那个漂亮网页。
